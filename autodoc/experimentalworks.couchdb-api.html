Overview


A simple CouchDB Wrapper




Public Variables and Functions



  

  

  
couch-add

  function

  
Usage: (couch-add url struct)


  
Add a given struct to the CouchDB at the given url.

  


  

  

  
couch-bulk-update

  function

  
Usage: (couch-bulk-update url xs)


  
Update multiple documents

  


  

  

  
couch-connection

  function

  
Usage: (couch-connection url method)


  
Creates a new connection to a CouchDB server. The used method can
be either :post, :get, :delete, or :put.

  


  

  

  
couch-delete

  function

  
Usage: (couch-delete url id)


  
Delete a document of given id from the CouchDB instance at url.

  


  

  

  
couch-get

  function

  
Usage: (couch-get url id)


  
Get a document with the given id from the DB.

  


  

  

  
couch-update

  function

  
Usage: (couch-update url id struct)


  
Update the given document id with the new struct.

  


  

  

  
couch-view

  function

  
Usage: (couch-view url design view)
       (couch-view url design view keys options)


  
Returns a view with proper key value mappings

  


  

  

  
with-conn

  macro

  
Usage: (with-conn bindings & body)


  
Assumes that the first binding is a connection and closes
the connection after executing the body.
